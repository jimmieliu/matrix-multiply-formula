# matrix-multiply-formula

Simple script to manipulate formula of matrix multiplication and other matrix manipulation ops


```
L = 2; D = 2

Q ∈ R(L, D)
| q00 q01 |
| q10 q11 |


K ∈ R(L, D) 
| k00 k01 |
| k10 k11 |

O ∈ R(L, 1)
K \sum_{i} Q_i
| k00(q00+q10)+k01(q01+q11) |
| k10(q00+q10)+k11(q01+q11) |
```
